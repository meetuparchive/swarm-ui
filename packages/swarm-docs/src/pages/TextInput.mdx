## TextInput
---
A wrapper for an html `<input>` element with an optional `<label>` tag.

<br/>
<br/>

### Snippet
---

Use this interactive snippet to try the component:
<br/>

```.jsx
<TextInput id="input" value="phife dawg two" />
```

<br/>
<br/>

### a11y
---
<br/>

Keyboard interactions: **n/a**

<br/>
Must supply a <b>label</b> html tag with an <b>`htmlFor`</b> React attribute that equals the value of the textInput's <b>id</b>.
